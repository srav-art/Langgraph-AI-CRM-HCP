## 🏗️ System Architecture

The system is designed using **5 LangGraphs**, each handling a specific responsibility in the AI CRM workflow:

### 1️⃣ Interaction Graph
Handles core HCP interactions:
- Log new interactions
- Edit previous interactions
- Store visit details

---

### 2️⃣ Intelligence Graph
Handles AI-driven decision making:
- Product recommendations
- Next-step suggestions

---

### 3️⃣ Insights Graph
Generates analytical insights from interaction history:
- Patterns in HCP interactions
- Summary insights

---

### 4️⃣ Follow-up Graph
Manages post-interaction planning:
- Suggests follow-up actions
- Recommends next engagement strategy

---

### 5️⃣ HCP Information Graph
Handles HCP profile-related queries:
- Fetch doctor/HCP details
- Retrieve contextual information

---
